# 💰 Finance App

Aplicação web para **controle de finanças pessoais**, permitindo registrar receitas e despesas, categorias e visualizar saldos e transações por mês.

O projeto foi desenvolvido com **React e Firebase**, utilizando Hooks para gerenciamento de estado e persistência de dados em banco.

---

## 🚀 Tecnologias utilizadas

- ReactJS
- JavaScript
- Firebase
- React Hooks
- CSS

---

## 🎯 Funcionalidades

- Cadastro de receitas
- Cadastro de despesas
- Filtro por mês
- Persistência de dados com Firebase
- Interface dinâmica com React

---

## 🧠 Conceitos aplicados

- Componentização em React
- Gerenciamento de estado com Hooks
- Manipulação de listas com `map()`
- Integração com backend (Firebase)
- Estruturação de projeto frontend
